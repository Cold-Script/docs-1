## Loadstring

<Info>Tracers require `Drawing` library.</Info>
```lua esp_lib.lua
local ESPLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/mstudio45/MS-ESP/refs/heads/main/source.lua"))()
```

After the ESP loads you can also access it like this:
```lua esp_from_getgenv.lua
getgenv().mstudio45.ESPLibrary
```
