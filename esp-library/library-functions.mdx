title: 'Library Functions'

### SetPrefix
<ParamField path="prefix" type="string" required>
  The prefix that is used in logging functions.
</ParamField>

```lua
ESPLibrary.SetPrefix("ESP")
```

### SetIsLoggingEnabled
<ParamField path="enabled" type="boolean" required>
    If `enabled` is `true`, info/warn/error messages will show up.
</ParamField>

```lua
ESPLibrary.SetIsLoggingEnabled(false)
```

### SetDebugEnabled
<ParamField path="enabled" type="boolean" required>
    If `enabled` is `true`, debug messages will show up.
</ParamField>

```lua
ESPLibrary.SetDebugEnabled(true)
```
